# Student_Attendence
📘 Student Attendance Viewer

A simple React-based web application to track and view student attendance records.

🚀 Features
View student attendance records
Clean and minimal UI
Built using modern React (hooks)
Fast development with Vite
🛠️ Tech Stack
Frontend: React
Build Tool: Vite
Styling: CSS
